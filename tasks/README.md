# Zadania dla uczestników

W tym katalogu będą znajdować się zadania praktyczne związane z tematyką współbieżności i wielowątkowości w Pythonie.

Planowane zadania:
- wykrywanie i naprawa race condition
- synchronizacja dostępu do zasobów współdzielonych
- bezpieczna komunikacja między wątkami
- poprawne zamykanie pracy wątków
